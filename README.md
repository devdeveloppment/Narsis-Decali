# NARSIS DECALI — Trading Premium

Plateforme de trading premium offrant des formations, des signaux forex et du copy trading. Basé au Togo et rayonnant sur toute l'Afrique de l'Ouest.

## 🚀 Fonctionnalités

- **Interface Premium** : Design moderne avec mode sombre, glassmorphisme et animations fluides.
- **Tableau de Bord Live** : Graphiques de trading interactifs mis à jour en temps réel.
- **Responsive Design** : Optimisé pour tous les appareils (Desktop, Tablette, Mobile).
- **Intégration WhatsApp** : Boutons de contact directs pour rejoindre la communauté et les formations.
- **SEO Optimisé** : Structure sémantique pour une meilleure visibilité sur les moteurs de recherche.

## 🛠️ Technologies Utilisées

- HTML5 / CSS3 (Vanilla CSS)
- JavaScript (Vanilla JS)
- FontAwesome 6 (Icônes)
- Swiper.js (Carrousels de témoignages)
- Canvas API (Graphiques temps réel)

## 📁 Structure du Projet

- `index.html` : Fichier principal contenant la structure, le style et la logique du site.
- `logo.png` : Logo officiel de la marque.

## 💻 Installation Locale

1. Clonez le dépôt :
   ```bash
   git clone https://github.com/devdeveloppment/Narsis-Decali.git
   ```
2. Ouvrez `index.html` dans votre navigateur.
